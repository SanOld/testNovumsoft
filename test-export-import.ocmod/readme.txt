Export/Import Tool (V1.0) for OpenCart 2.x
===========================================


Тестовое задание для Novum Software============================


Модуль экспорта импорта списка продуктов============



Для OpenCart admin backend, do the following steps:


Шаг 1

	Перейти в Extensions > Extension Installer


Шаг 2

	Скачать test-export-import.ocmod.zip



Шаг 3
	Перейти Extensions > Modifications
	В списке должен быть "test Export (V1.0) for OpenCart 2.x


Шаг 4 

	Нажать кнопку "Обновить" (справа вверху)


Шаг 5 

	Перейти System > Users > User Group > Edit Administrator

Шаг 6 
	Разрешить доступ к 'extensions /export_import'


В меню Extensions должен быть пункт Export/Import xls
